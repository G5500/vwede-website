# Vwede Software Company Website

This is the official static website for **Vwede Software Company**, showcasing our services, about us, and contact information.

## 🌐 Live Demo

You can view the live website here: [Vwede Website](https://G5500.github.io/vwede-website/)

## 📂 Project Structure

- `index.html`: Main HTML file for the website.

## 🚀 Deployment

This website is deployed using **GitHub Pages**. To update the website:

1. Make changes to the `index.html` file.
2. Commit and push the changes to the `main` branch.
3. The website will automatically update at [https://G5500.github.io/vwede-website/](https://G5500.github.io/vwede-website/)
